Color Finder

This page compiles standardized web color names as defined by the W3C CSS Color Module Level 4, which incorporates and extends the original HTML4 color set. The 148 standard colors are presented below in tabular form and represent a canonical system for browser-supported color naming.

A custom input panel allows users to explore any color through hexadecimal codes, extending beyond the standardized set. Colors not included in this specification are labeled as Unspecified. RGB values correspond to the additive color model used in digital displays, whereas CMYK values represent approximate conversions to the subtractive model used in print.
